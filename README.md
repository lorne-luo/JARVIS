JARVIS
===============================

version number: 0.0.1
author: Lorne Luo

Overview
--------

Notification micro services by SMS, EMail, Telegram, Weixin

Installation / Usage
--------------------

To install use pip:

    $ pip install jarvis


Or clone the repo:

    $ git clone https://github.com/lorne-luo/jarvis.git
    $ python setup.py install


How to run
--------------------
    python server.py
